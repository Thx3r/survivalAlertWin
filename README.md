# survivalAlertWin
Simple python3 script which mail alert admin about HDD and user logon for Windows 

Features : 
- check Hdd Health (based on wmic command)
- check free space Hdd
- check who is logon and mail if not in whitelist
- Messages Box alerts
- Full customizable
